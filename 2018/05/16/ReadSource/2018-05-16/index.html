BuiltinMath 是swift 内置的函数库，函数除了方法以外，调用方式其实都是一样的，先来看cos，

（1）Float(_bits:) 方法，位于FloatingPointTypes.swift 文件

Float(_bits:) 函数

（2）Builtin.FPIEEE32 类型
x._value 的值就是Builtin.sitofp_Int64_FPIEEE32(zero._value)
Builtin.FPIEEE32 是什么类型呢？ 在Builtins.cpp文件里有

public func cos( x: Double) -> Double {
return Double(_bits: Builtin.int_cos_FPIEEE64(x._value))
}

```
两个函数只有参数类型的差别，区别的是Float 调用int_cos_FPIEEE32函数，double调用int_cos_FPIEEE64函数

（5）其他函数
*_sin 正弦函数
*_cos 余弦函数
*exp 指数函数
*exp2 以自然常数2为底的指数函数
*_log 对数函数
*_log10 计算以10 为底的对数值
*_log2 计算以2 为底的对数值
*_nearbyint 通过使用当前舍入方向，舍入参数为浮点格式的一个整数值，整数。
*_rint 将浮点值舍入到最接近的整数
注意 _nearbyint和 _rint 相似，区别只是返回值的不同.
